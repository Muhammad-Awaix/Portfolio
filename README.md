## 🚀 Featured Projects

### 1. 📄 **CV Job Matcher**
AI-powered resume analyzer that matches CVs against job descriptions using LangChain + Groq. Deployed on Streamlit Cloud with structured output parsing.

**Stack:** LangChain • Groq • Streamlit • Pydantic  
**[→ View on GitHub](https://github.com/Muhammad-Awaix)**

### 2. 🧠 **LangGraph ReAct Agent**
Production-ready AI agent built with LangGraph's ReAct architecture. FastAPI backend + glassmorphism Flask frontend. Handles multi-step reasoning tasks.

**Stack:** LangGraph • FastAPI • Flask • ReAct  
**[→ View on GitHub](https://github.com/Muhammad-Awaix/ai-agent-langgraph)**

### 3. 📰 **AI News Digest Dashboard**
Automated news aggregation and summarization pipeline using feedparser + Groq API. Delivers categorized AI/tech news digest with real-time updates.

**Stack:** feedparser • Groq • Python • Dashboard  
**[→ View on GitHub](https://github.com/Muhammad-Awaix)**

### 4. 🎥 **YouTube RAG Assistant**
RAG pipeline that ingests YouTube transcripts and answers questions about video content. Built with FastAPI, LangChain, FAISS vector store, and Groq LLM.

**Stack:** RAG • FAISS • LangChain • FastAPI  
**[→ View on GitHub](https://github.com/Muhammad-Awaix)**

### 5. 📚 **AI Book Recommendation System**
Semantic book recommendation engine leveraging vector embeddings and LLM-powered explanations. Finds books based on mood, themes, and reading preferences.

**Stack:** Vector Embeddings • LLM • Semantic Search  
**[→ View on GitHub](https://github.com/Muhammad-Awaix)**
